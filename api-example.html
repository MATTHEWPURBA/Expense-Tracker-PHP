üí° What to Look For

In Network Tab

  • Name: The API endpoint being called
  • Method: HTTP method (GET, POST, etc.)
  • Status: 200 = success, 404 = not found, etc.
  • Time: How long the request took
  • Response: Click to see the full JSON response

In Console Tab

  • üåê API Request: Details about the request
  • üì• Response: Response data and timing
  • üêõ Debug Information: Route, controller, and queries
  • üíæ Database Queries: All SQL queries executed

In Response Data

  • success: true/false
  • data: The actual response data
  • _debug: Debug information (if enabled)
  • queries: List of SQL queries executed
  • performance: Execution time and memory usage
